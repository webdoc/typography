.index class

The .index class defines a list of inline-blocks. It can be extended to make divided_index menus, grids of thumbs or icons, or fancy definition lists.

The .thumbs class can use background sprites or individual background images. If you opt for a sprites, the image file should be arranged vertically with sufficient transparent space between the icons to accommodate a line or two of text. For example: tools.png
